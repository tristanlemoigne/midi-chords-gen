# MIDI EMITTER

This project generates procedural chords progressions.

- Mac only (CoreMidi, CoreFoundation)

### Deps

- rtmidi
- nlohmann-json

#### Setup & Build

`brew install rtmidi`

`brew install nlohmann-json`

`cmake .`

`cmake --build .`

### Run Program

`bin/main`
